## 🚀 快速開始

### 新手上路
1. 閱讀 `PROJECT.md` 了解專案目標
2. 查看 `LAYOUT.md` 了解版面架構
3. 參考 `DESIGN_SYSTEM.md` 學習設計規範

### AI Agent 協作
1. 先讀根目錄的 `agents.MD`
2. 查看 `WORKFLOW.md` 了解協作流程
3. 依照 `PROJECT.md` 的實作階段執行

### 開發者
1. 根目錄 `DEVELOPMENT.md` - 環境設定
2. `DESIGN_SYSTEM.md` - SCSS 變數系統
3. `LAYOUT.md` - 實作版面與導航
